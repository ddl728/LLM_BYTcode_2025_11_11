# 时间：2025_11_11
主要功能：
  # 1.场景解析器
    ## (1)获取机器人基座的坐标和四元数数据、工件的坐标、四元数数据及包围盒尺寸。(四元数以[X、Y、Z、W]顺序输出；并且模型的真实包围盒尺寸是通过方法获取的2倍，需要x2才对)
    ## (2)通过ScocketServer将数据以String的形式输出到python端
  # 2.机器人
    ## (1) 通过python端将工件位姿(坐标+四元数)转换成相对基座的相对位姿，并且将转换后的数据通过脚本驱动机器人运动到指定工件位置
       (2)在原有抓取工件的基础上添加了复位home的动作，需要注意的是，通过相对基坐标计算得到的[X、Y、Z、W]运动时，末端执行件朝向有时会错误。所以只要留工件的位置 (X, Y, Z)，但是强制指定一个固定的抓取方向 ([0,0,1,0])（例如始终垂直向下）,因为在工业机器人应用中，我们通常不直接使用工件的姿态作为抓取姿态。但复位的相对四元数是通过直接计算然后运动的。
